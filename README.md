# FastAPI Job Portal

A backend Job Portal application built using FastAPI.

## Features
- User Registration & Login
- Job Posting
- Apply for Jobs
- REST API Endpoints

## Tech Stack
- Python
- FastAPI
- SQLAlchemy
- SQLite
- Uvicorn

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the server:
   uvicorn main:app --reload

3. Open in browser:
   http://127.0.0.1:8000/docs
